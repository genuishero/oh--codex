# Official Docs Check

Use this skill when the answer should be verified against official documentation.

## What to do

1. Prefer official docs and primary sources over memory.
2. If the information is time-sensitive, verify it before answering.
3. Answer the direct question first.
4. Distinguish verified facts from local inference.
5. Include the smallest useful set of source links.

## Output shape

- Direct answer
- Verified source links
- Short note on any assumptions or local constraints
